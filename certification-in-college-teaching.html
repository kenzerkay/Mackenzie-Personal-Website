Certification in College Teaching (Work In Progress)

About the Program

The Certification in College Teaching (CCT) is a formal certification program at Michigan State University that gives graduate students and postdocs the opportunity to learn about and develop teaching strategies. This program focuses on completing 5 core competencies and a teaching philosophy statement to show development of key teaching principles.

5 Core Competencies:

  • Core Competency 1: Developing Discipline-Related Teaching Strategies
  • Core Competency 2: Creating Effective Learning Environments
  • Core Competency 3: Incorporating Technology in your Teaching
  • Core Competency 4: Understanding the University Context
  • Core Competency 5: Assessing Student Learning

For each of these competencies I will include artifacts and materials developed from the activities I used to completed these competencies.
